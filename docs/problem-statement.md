# Problem Statement — CalmPulse BP

Saya hidup dengan hipertensi dan sering mengalami momen ketika tubuh sebenarnya sudah memberi sinyal untuk berhenti sejenak, tetapi rutinitas kerja membuat sinyal itu terlewat. CalmPulse BP dirancang sebagai aplikasi Garmin watch-only yang memberi intervensi mikro berupa haptic nudge dan sesi napas 60 detik ketika kombinasi sinyal sederhana (tren heart rate relatif terhadap baseline pribadi dan periode inaktivitas) terdeteksi.

Solusi ini tidak mengklaim diagnosis tekanan darah. Fokusnya adalah membantu pengguna hipertensi muda meningkatkan self-awareness, membangun kebiasaan jeda sehat, dan mencatat refleksi singkat setelah intervensi. Hipotesis dampak MVP: pengguna lebih cepat merespons tanda stres awal, menyelesaikan lebih banyak sesi regulasi napas, dan menurunkan frekuensi momen “terlambat sadar” dalam aktivitas harian.
